Guide d'installation

Installer labCE v5.3

Suivez ce guide pas à pas pour installer labCE sur Windows, avec ou sans le driver NI-DAQmx.

Méthode recommandée — 1 double-clic

INSTALLER.bat installe NI-DAQmx, copie labCE et crée le raccourci bureau automatiquement. Prévoir 10 à 30 min.

✅ Vérifier avant de lancer

Package ZIP complètement extrait (pas lancé depuis l'archive)
Dossier NI-DAQmx/ présent à côté de INSTALLER.bat
Fichier labCE_app.exe présent dans le dossier
Droits administrateur disponibles sur ce PC
Au moins 5 GB d'espace disque libre
Windows 10 ou 11 (64 bits)
0 / 6 prérequis validés0%

📋 Étapes détaillées

1
Extraire le package ZIP complet

Clic-droit sur labCE_v5.3_Package.zip → "Extraire tout" dans un dossier sans accent ni espace (ex. C:\labCE).

Ne pas lancer depuis le ZIP

Windows bloque les exécutables lancés directement depuis une archive. Extraire d'abord.

2
Vérifier la structure du dossier
Structure attendue
labCE_v5.3_Package/ ├── labCE_app.exe ← application ├── INSTALLER.bat ← à double-cliquer ├── UPDATE.bat ├── config.ini ├── Logo_LabCE.ico └── NI-DAQmx/ ← driver NI offline ├── ni-daqmx_25.8.1_offline.iso (3.2 GB) └── *.nipkg
3
Double-cliquer sur INSTALLER.bat

Si une fenêtre UAC apparaît → cliquer Oui. Si rien ne se passe → clic-droit → "Exécuter en tant qu'administrateur".

Console — sortie attendue

[OK] nipkg.exe trouvé [2/4] Vérification NI-DAQmx... Montage de l'image disque... Installation NI-DAQmx... (5 à 20 min) [OK] NI-DAQmx installé avec succès. [3/4] Installation de labCE... 4
Redémarrer si demandé (code 3010)

Si la console affiche [INFO] Un redémarrage sera nécessaire, redémarrez Windows normalement avant de lancer labCE. Ce n'est pas une erreur.

5
Configurer config.ini (optionnel)

Le fichier est copié dans %LOCALAPPDATA%\labCE\. Adaptez le nom du device NI à votre poste.

config.ini
[NI] device = Dev1 ; nom tel qu'affiché dans NI MAX has_ni = true ; false = mode simulation [Channels] effort_ai = ai0 course_ai = ctr0 [Acquisition] frequency_hz = 50 [Sensors] sensitivity_effort_nv = 1.0 sensitivity_course_mmv = 1.0
✓
Lancer labCE depuis le raccourci bureau

Structure requise

[OK] labCE mis a jour avec succes. Lancez l'application depuis le raccourci sur le bureau.

[ERREUR] La copie a échoué

labCE est encore ouvert → Gestionnaire des tâches → fin de tâche labCE_app.exe → relancer UPDATE.bat.

✓
Vérifier la version dans la barre de titre

Problèmes fréquents

❌[ERREUR] labCE_app.exe introuvable
▼

INSTALLER.bat ou UPDATE.bat doit être dans le même dossier que labCE_app.exe. Ne pas l'exécuter depuis un raccourci ou un autre emplacement.

⚠️NI-DAQmx non disponible / device non détecté
▼

Normal sans matériel NI — labCE fonctionne en simulation. Si vous avez du matériel :

  • Vérifier que NI-DAQmx est installé (Panneau de configuration → Programmes)
  • Ouvrir NI MAX — le device doit être visible
  • Vérifier le champ device = Dev1 dans config.ini
  • Cliquer "🔄 Rafraîchir" dans labCE
  • Redémarrer si NI-DAQmx vient d'être installé
🔒Windows SmartScreen / Antivirus bloque l'exe
▼
  1. Clic-droit sur labCE_app.exe → Propriétés
  2. En bas → cocher "Débloquer" → OK
  3. Ou : message SmartScreen → "Informations complémentaires" → "Exécuter quand même"
🐍ModuleNotFoundError (sources Python)
▼
CMD
pip install --upgrade numpy matplotlib openpyxl pillow scipy nidaqmx
🔴Code retour 3010 après installation NI-DAQmx
▼

Pas une erreur

Code 3010 = installation réussie, redémarrage requis. Redémarrez Windows normalement. labCE fonctionnera après.
